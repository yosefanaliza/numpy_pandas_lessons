# NumPy Lessons

This directory contains 20 comprehensive NumPy lessons organized into separate files.

## Lesson Structure

Each lesson is self-contained and can be run independently:

### Basics (Lessons 1-5)
- **Lesson 1**: Creating NumPy Arrays
- **Lesson 2**: Array Creation Functions
- **Lesson 3**: Array Attributes and Properties
- **Lesson 4**: Array Indexing and Slicing
- **Lesson 5**: Reshaping and Transposing Arrays

### Operations (Lessons 6-9)
- **Lesson 6**: Basic Array Operations
- **Lesson 7**: Mathematical Functions
- **Lesson 8**: Statistical Functions
- **Lesson 9**: Array Manipulation

### Advanced Topics (Lessons 10-14)
- **Lesson 10**: Linear Algebra Operations
- **Lesson 11**: Random Number Generation
- **Lesson 12**: Broadcasting in Detail
- **Lesson 13**: Advanced Indexing Techniques
- **Lesson 14**: Sorting and Searching

### Optimization & Data (Lessons 15-17)
- **Lesson 15**: Performance Tips and Vectorization
- **Lesson 16**: Working with Data Types
- **Lesson 17**: Saving and Loading Arrays

### Special Arrays & Examples (Lessons 18-20)
- **Lesson 18**: Structured Arrays
- **Lesson 19**: Masked Arrays (Handling Missing Data)
- **Lesson 20**: Practical Examples

## Running Lessons

### Run Individual Lesson
```bash
python lesson01_numpy_arrays.py
python lesson05_array_reshaping.py
# etc.
```

### Run All Lessons
```bash
python run_all.py
```

### Import as Module
```python
from numpy_lessons import lesson01_numpy_arrays
lesson01_numpy_arrays.main()
```

## Requirements
- numpy

Install with:
```bash
pip install numpy
```
